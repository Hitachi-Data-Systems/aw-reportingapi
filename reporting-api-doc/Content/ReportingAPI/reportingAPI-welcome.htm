Welcome

Welcome to the Hitachi Content Platform Anywhere (HCP Anywhere) Reporting API Reference Guide. The Reporting API can be used to generate reports, so that you can track:

• User activity

•Profile activity

•System activity

Intended audience

The HCP Anywhere Reporting API Reference Guide is for HCP Anywhere administrators who want to generate reports about their users and their HCP Anywhere system.
