HCP Anywhere Reporting API Reference Guide

Accessing documentation

The HCP Anywhere documents can be accessed from a variety of places, including the Management Console, GitHub, and the Hitachi Vantara knowledge site.

For the File Sync and Share API, there are a variety of resources at your disposal. In addition to the reference guide for the File Sync and Share API that's available in the Management Console, there is also a Java software developers kit (SDK) on the Maven Central Repository and a sample program that uses the SDK on GitHub.

Maven—To access the Maven Central Repository, see http://search.maven.org. If you don’t know how to use the Maven Central Repository, see http://central.sonatype.org/.

GitHub—To access GitHub, see https://github.com/Hitachi-Vantara/anywhere-fss-sdk.

To access the HCP Anywhere documentation in the Management Console:

In the Management Console, you can access the documents you need to help you administer HCP Anywhere. The following documents are available in the Management Console:

•HCP Anywhere Administrator Help

•Installing an HCP Anywhere System - Final On-site Setup

•Deploying an HCP Anywhere VM System

•Third-party Copyrights and Licenses

•HCP Anywhere File Sync and Share API Reference Guide

•HCP Anywhere Reporting API Reference Guide

1.In the top right hand corner of the Management Console, mouse over the documentation icon (  ).

2.In the menu, click on the document you want.

To access the HCP Anywhere documentation on the Hitachi Vantara knowledge site:

The Hitachi Vantara knowledge site is the ideal place to access documentation because it includes all of the documents for the release in one place.
