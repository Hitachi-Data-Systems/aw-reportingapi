Creating a profile

Once you've configured Active Directory or SAML, you can give users access to HCP Anywhere. You do this by creating profiles in the HCP Anywhere Management Console. In each profile, you configure:

•The users in the profile.

•The list of features and HCP Anywhere applications the users in the profile have access to.

Important: If a user belongs to multiple profiles, that user gets a combination of the most permissive settings from each of their profiles.
