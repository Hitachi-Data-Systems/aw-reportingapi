Response codes

The table below describes the possible response codes returned from a request for a report.

HTTP Response Code Error string Meaning
200 OK Success. The report is complete, and if it's paged, you have reached the last page.
206 Partial Content Partial content (More pages are available).
400 Bad Request Bad input parameter. Error message specifies which input parameter is faulty and why.
401 Unauthorized Bad or expired token.
403 Forbidden Request is forbidden. You are requesting a report and do not have the without the correct permissions.
404 Not Found Report was not found.
501 Not Implemented Feature Not Implemented.
503 Service Unavailable The service you are requesting is currently unavailable.
5XX N/A Server error.
