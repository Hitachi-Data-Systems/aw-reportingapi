Accept and decline team folder storage requests

When you enable team folders for a profile, you set the maxim space permitted for each team folder in the Threshold for requests field. For example, if you set the threshold to 10 GB, then each team folder created by a user from that profile can have up to 10 GB of space

If a user sets the total storage quota of a team folder above this quota limit, the space is not approved or declined immediately. Instead, a request is sent to you for the additional space. You can then accept or decline the request.  You can see all the team folder requests on the Team Folders page in the Requests tab.

To accept and decline team folder requests:

1.In the top-level menu in the Management Console, mouse over File Sync and Share.

2.In the menu, click on Team Folders.

3.On the left side of the page, click on Requests.

The team folder requests are listed. For each team folder, the user's requested quota is listed in the Requested Quota column and the current quota capacity of the team folder is listed in the Allocated Quota column.

4.For each team folder request you want to answer, do one of these:

oAnswer the request on the dashboard by selecting the accept icon ( ) or decline icon ( ) on the right side of the team folder's row.

oClick on the team folder. Then click on the Approve button to accept the request or the Decline button to decline the request.

Tip:  You can also change the quota of any request by clicking on the team folder you want and typing a new a new quota in the Quota field. Then click on Approve to set the new quota for the team folder.
