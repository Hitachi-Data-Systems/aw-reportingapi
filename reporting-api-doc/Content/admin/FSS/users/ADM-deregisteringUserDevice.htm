HCP Anywhere Reporting API Reference Guide

Deregistering user devices

Deregistering a device removes it from the system. When a device is deregistered, a user must reregister the device before it can communicate with the HCP Anywhere system again. You should deregister a device if the device on which the device is installed is lost or stolen.

Desktop devices

When you deregister a desktop device, you can choose if you want to Delete user files. When you select this option:

•The next time that the desktop application connects to the system, all files are deleted from the HCP Anywhere folder on the user's computer. This includes any files that have not yet been stored in HCP Anywhere.

•HCP Anywhere does not delete the user's files from HCP or from any of the other devices that the user has registered.

Mobile devices

When you deregister an HCP Anywhere mobile device, all the HCP Anywhere data on the device is deleted.

Deregistering a device removes it from the system. When a device is deregistered, a user must reregister the device before it can communicate with the HCP Anywhere system again.

Note: HCP Anywhere can be configured to automatically deregister devices after a certain number of days of inactivity. Automatic deregistration is set on a profile basis. For information on configuring automatic deregistration, see Editing profile settings.

To deregister a user's device:

1.In the top-level menu, mouse over Access.

2.In the menu, click on Registered Users.

3.In the list of registered users, click on the row for the user you want.

4.On the Overview tab, click on the deregister control ( ) for the device you want to deregister.

A confirming prompt appears. If you are deregistering a computer, the prompt asks you whether you want to delete all files from the HCP Anywhere folder on the computer in addition to deregistering the computer.

5.Conditionally, do one of these:

oTo delete the user's files from the device, leave the Delete user files option selected.

oTo leave the user's files on the device, deselect the Delete user files option.

6.Click on the Deregister device button.
