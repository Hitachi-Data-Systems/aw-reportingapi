Install and manage Edge

Edge is an HCP Anywhere device that is designed to take the data on your Windows Servers (network-attached storage) and back it up to HCP Anywhere (cloud storage). Edge can back up data from almost any folder on a Windows Server, but is typically used to back up user home folders, group or departmental folders, and application folders. Edge can be used to:

• Transition users and data from legacy NAS to HCP Anywhere

•Back up user and application data from multiple remote sites into one central cloud system

•Intelligently manage space on your Windows Servers using the Edge cache
