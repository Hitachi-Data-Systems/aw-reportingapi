Auditing registered users

HCP Anywhere administrators that have the Audit role can use the Registered Users page to view an individual registered user's files and file synchronization and sharing activity.

To audit events and files for an individual user:

1.In the top-level menu, mouse over Access.

2.In the menu, click on Registered Users.

3.In the list of registered users, click on the audit control ( ) for the user you want to audit.

4.In the Audit User window:

oTo view user events, click on the Activity tab. For more information, see Viewing user activity.

oTo view the user's files, click on the Files tab. For more information, see Viewing user files.
