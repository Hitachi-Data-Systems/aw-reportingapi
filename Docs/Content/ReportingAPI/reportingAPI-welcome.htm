Welcome

Welcome to the Hitachi Content Platform Anywhere (HCP Anywhere) Reporting API Reference Guide. This guide will teach you how to generate reports so that you can track:

• User activity

•Profile activity

•System activity

This information is also available in the HCP Anywhere Administrative Help.

Intended audience

The HCP Anywhere Reporting API Reference Guide is for HCP Anywhere administrators who want to customize reports about their users and HCP Anywhere system.
